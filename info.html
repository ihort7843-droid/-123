Про додаток

Movie Finder — це простий навчальний проєкт, який використовує API для пошуку фільмів.

Як це працює?

Коли ти вводиш назву фільму, додаток звертається до API і отримує список результатів.

Навіщо це?

Проєкт показує, як працює JavaScript з API, і як можна створити власний веб-сервіс.
